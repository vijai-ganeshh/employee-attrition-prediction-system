# Employee Attrition Prediction (Machine Learning)

This project predicts employee attrition using the IBM HR Analytics dataset.

## Dataset
IBM HR Analytics Employee Attrition & Performance (Kaggle)

## Features
- Feature engineering and preprocessing
- Logistic Regression and Random Forest models
- Evaluation using accuracy, precision, recall
- Interactive Streamlit dashboard

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
